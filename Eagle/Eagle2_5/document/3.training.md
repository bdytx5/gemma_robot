### 🟢 Launching Stage-1 Training
Typically, you only need to train on top of our pre-trained model. If you want to start from a base LLM and a Vision Encoder, 
please refer to this script: [stage-1 training](./stage_1_training.md)

### 🟢 Launching Finetuning

Run the following command to start training:

```bash
GPUS=8 bash shell/train_stage2.sh 1 work_dirs/eagle2.5_debug
```

If everything goes smoothly, your first step log should look like this:


### Notes
If you use internal job watchers to auto-cancel failing runs, ensure your training scripts do not print ignorable exceptions with full Traceback; otherwise watchers may incorrectly terminate jobs.
